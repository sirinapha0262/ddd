🤖 ข้อ 1: ASM Chart – เครื่องซักผ้าหยอดเหรียญ

📖 ASM Chart คืออะไร?

ASM (Algorithmic State Machine) Chart คือ แผนภาพที่แสดงการทำงานของระบบดิจิทัลแบบ Sequential มีองค์ประกอบหลัก 3 ส่วน:

⬜ State Box
รูปสี่เหลี่ยม — แสดงสถานะปัจจุบันของระบบ เช่น IDLE, WASHING, DRYING
◇ Decision Box
รูปเพชร — แสดงเงื่อนไข Input มีแขนง YES / NO เช่น Coin_10 = 1?
⬭ Output Box
รูปวงรี — แสดงสัญญาณ Output ที่ส่งออก เช่น Wash=1, Dry=1

🧺 โจทย์: เครื่องซักผ้าหยอดเหรียญ

📋 เงื่อนไขของระบบ
  • 1️⃣ Input: Coin_10 — รับเหรียญ 10 บาทต่อครั้ง สะสมจนครบ 40 บาท (4 ครั้ง)
  • 2️⃣ Output: Wash = 1 — เมื่อครบ 40฿ เริ่มซักผ้า 1 รอบ รอสัญญาณ Timer
  • 3️⃣ Input: Timer_Done — สัญญาณจาก Timer หลังซักเสร็จ → เริ่มปั่นแห้ง
  • 4️⃣ Output: Dry = 1 — ปั่นแห้งจนเสร็จสิ้น
  • 5️⃣ กลับสู่สถานะ IDLE รอรับเหรียญรอบใหม่
สถานะทั้งหมด 6 States: S0(IDLE) → S1(10฿) → S2(20฿) → S3(30฿) → S4(WASH) → S5(DRY) → S0

❓ สัจนิรันดร์ (Tautology) คืออะไร?

📖 นิยาม
สูตรที่เป็น จริงทุกกรณี ไม่ว่าจะแทนค่าตัวแปรอย่างไร เรียกว่า "สัจนิรันดร์"
✅ ตัวอย่าง Tautology
  • p ∨ ¬p
  • p → p
  • [(p∨q)∧¬p] → q ✓
❌ ไม่ใช่ Tautology
  • p ∧ q
  • p → q
  • p ∨ q
🔄 ข้อ 3-4: กฎสมมูล & อาร์กิวเมนต์

📝 พิสูจน์: ¬(p → q) ≡ p ∧ ¬q

🎯 เป้าหมาย
แปลงฝั่งซ้าย ¬(p → q) ให้ได้ p ∧ ¬q โดยใช้กฎสมมูล
¬(p → q)
เริ่มต้น
≡ ¬(¬p ∨ q)
Implication Law: p→q ≡ ¬p∨q
≡ ¬(¬p) ∧ ¬q
De Morgan: ¬(A∨B) ≡ ¬A∧¬B
≡ p ∧ ¬q
Double Negation: ¬(¬p) ≡ p ✅
✅ พิสูจน์สำเร็จ!
ใช้ 3 ขั้น: Implication → De Morgan → Double Negation

📚 กฎสมมูลที่ต้องจำ

1️⃣ Implication Law
p → q ≡ ¬p ∨ q
2️⃣ De Morgan's Laws
¬(p∧q) ≡ ¬p∨¬q
¬(p∨q) ≡ ¬p∧¬q
3️⃣ Double Negation
¬(¬p) ≡ p
4️⃣ Modus Ponens
p→q, p ⊢ q
5️⃣ Modus Tollens
p→q, ¬q ⊢ ¬p
6️⃣ Contrapositive
p→q ≡ ¬q→¬p

✅ ข้อ 6: Tautology Checker

🐛 โค้ดที่มีบั๊ก
def is_tautology(): results = [] for p in [True, False]: for q in [True, False]: lhs = p and ((not p) or q) # ❌ ผิด! ควรเป็น (p∨q)∧¬p expression = (not lhs) or q results.append(expression) return all(results) print(f"Is it a tautology? {is_tautology()}")
🐛 ปัญหา
lhs ควรเป็น (p or q) and (not p) ไม่ใช่ p and ((not p) or q)
✅ โค้ดที่ถูกต้อง
def is_tautology(): results = [] for p in [True, False]: for q in [True, False]: lhs = (p or q) and (not p) # ✅ (p∨q) ∧ ¬p expression = (not lhs) or q # ✅ lhs → q results.append(expression) return all(results) print(f"Is it a tautology? {is_tautology()}") # Output: Is it a tautology? True
🔍 อธิบาย
all(results) คืน True เมื่อทุกค่าใน list เป็น True = สูตรเป็น Tautology

🔢 ข้อ 7: Basic Proof Logic – Prime Numbers

🎯 ทดสอบ
(Prime(n) AND n > 2) → IsOdd(n)
"ถ้า n เป็นเฉพาะและ > 2 แล้ว n ต้องเป็นเลขคี่"
def is_prime(n): if n < 2: return False for i in range(2, int(n**0.5) + 1): if n % i == 0: return False return True found_counterexample = False for n in range(1, 101): condition = is_prime(n) and n > 2 conclusion = n % 2 != 0 if condition and not conclusion: print(f"Counterexample found: {n}") found_counterexample = True break if not found_counterexample: print("Logic holds for n=1 to 100") # Output: Logic holds for n=1 to 100
